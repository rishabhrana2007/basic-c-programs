Basic C Programs Repository

This repository contains foundational C programs focused on core control-flow concepts. It is designed for beginners who want clean, minimal, and logically structured examples instead of bloated or outdated code.

* Included Topics

Conditional statements (if, if-else, nested conditions)

Looping constructs (for, while, do-while)

Decision-making using switch

Basic input/output handling

Simple mathematical and logical operations
